## 🕹️ 玩法

1. 点开所有不存在 💣 的方块格子则【胜利】
2. 踩到 💣 则【失败】
3. 鼠标左键单击点开格子
4. 点开的格子周围如果没有炸弹则会继续打开旁边的格子，直至周围存在 💣
5. 鼠标右键可以给格子标 🚩，表示该方格是找到的 💣
6. 双击可以打开周围除了标 🚩 的格子
